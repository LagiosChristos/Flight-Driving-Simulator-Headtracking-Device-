# Intro

This project is a DIY head-tracking device built using an Arduino Micro and a BNO055 absolute orientation sensor in combination with OpenTrack software, designed for use in flight and driving simulators. It captures real-time head movement and translates it into in-game camera control, providing a more immersive and intuitive experience.

Instructions on how to get started are provided in PDF format within "Instructions" folder.

# Hardware and Software

**Hardware**

* 1x Arduino Micro
* 1x BNO055 9-Axis IMU
* 4x Jumper Wires
* 1x Case
* 1x Velcro

**Software**

* Arduino IDE
* OpenTrack
